== Description ==

This plugin allow you to set discounts (in percents) for each user role.
this discount would be added to all products with no restriction.
The plugin is simple you just set the discount and all is ready. enjoy!!


A few notes about this plugin:

*   The plugin not add new roles
*   If you don't set a discount the plugin will show the regular price
*   Don't work over sale price

    I'm working to add support for sale price and variations, you can help me to get this sooner please help me with donations'

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Woocommerce->Settings->Products->Prices by Role
